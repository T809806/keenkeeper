1. KeenKeeper

A personal relationship management web app to track friendships. Users can view friends, check details, and log interactions through a simple timeline system.

2. Technologies Used

i. Next.js 
ii. React
iii. JavaScript
iv. Tailwind CSS
 v. JSON Data 


3. Key Features

i. Friend Details Page
- Displays friend profile, status and email
- 2 column responsive layout 

ii.  Quick Check In System
- Call / Text / Video buttons
- Real-time toast notification on each action

iii. Friend Tracking System
- Shows days since last contact
- Next due date display

#  KeenKeeper

## Overview
KeenKeeper is a personal relationship management web app that helps users track friendships and interactions in a simple and structured way.

## Technologies Used
- Next.js
- React
- JavaScript
- Tailwind CSS
- JSON Data 

---

##  Key Features

###  Friend Details Page
- Displays friend profile, status, and email
- Clean 2-column responsive layout
- User-friendly UI

---

###  Quick Check-In System
- Call / Text / Video interaction buttons
- Real-time toast notifications on actions
- Smooth UX feedback system

---

###  Friend Tracking System
- Shows days since last contact
- Displays next due date for interaction
- Helps maintain consistent communication

---

## How to Run Locally

```bash
npm install

```
